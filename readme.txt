Subscription-based WHOIS data distribution platform with admin dashboard, customer portal, and Stripe payments.

== Description ==

WHOIS CRM is a comprehensive WordPress plugin for managing and distributing WHOIS domain ownership data through a subscription-based model.

**Features:**

* Admin dashboard for uploading and managing WHOIS datasets
* Customer portal with subscription-based data access
* Stripe payment integration with recurring billing
* Secure file download with authorization checks
* Email notifications and PDF invoicing
* REST API for premium customers
* Reports and analytics
* Activity logging and security

== Installation ==

1. Upload the `whois-crm` folder to `/wp-content/plugins/`
2. Run `composer install` in the plugin directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to WHOIS CRM → Settings to configure Stripe API keys
5. Create the required WordPress pages with shortcodes (see documentation)
